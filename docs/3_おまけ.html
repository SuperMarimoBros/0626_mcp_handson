LLMを活用したMCP開発

🤖 Building MCP with LLMs

ClaudeなどのLLMを使用してMCPサーバーを効率的に開発する方法を解説します。

📖 以下の手順は、https://modelcontextprotocol.io/tutorials/building-mcp-with-llms の内容を元に作成したものとなります

1

オリジナルの手順の理解

まず、ClaudeがMCPを理解できるよう必要なドキュメントを準備。

  • • MCP完全ドキュメントをコピー
  • • Python SDKのREADMEをコピー
  • • これらの文書をClaudeとの会話に貼り付け

ドキュメントを提出したら、どのようなサーバーを構築したいのかをClaudeに明確に説明。

具体的に以下の点を明記する。

  • • サーバーが公開するリソース
  • • どのようなツールを提供するのか
  • • 提供されるべきプロンプト
  • • どのような外部システムとやり取りする必要があるか
2

上記を加味してプロンプトのみで依頼

プロンプト例
参考文献に従って以下の機能を持つMCPサーバーを作成してください： # 機能 - MCPサーバー名は「postgressql-mcp-server」 - ローカル環境のPostgreSQLデータベースに接続 - テーブルスキーマをリソースとして公開 - 読み取り専用のSQLクエリを実行するツールを提供 - 一般的なデータ分析タスク用のプロンプトを含める # 参考文献 - https://modelcontextprotocol.io/llms-full.txt - https://github.com/modelcontextprotocol/python-sdk

NOTE: Github Copilot Agent, Cursor, Clineなどのソースエディタ内のコード生成Agentで作成した方が作成効率が良さそうです。2025/06時点では、Claude 4が良さげです。
